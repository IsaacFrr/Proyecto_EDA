# Datasets del proyecto

Los CSVs no están incluidos en el repositorio por su tamaño.
Descargar manualmente desde las fuentes oficiales:

## Stack Overflow Developer Survey 2021
- URL: https://survey.stackoverflow.co/2021
- Archivo a renombrar: `survey_results_public.csv` → `survey_results_2021.csv`

## Stack Overflow Developer Survey 2025
- URL: https://survey.stackoverflow.co/2025
- Archivo a renombrar: `survey_results_public.csv` → `survey_results_2025.csv`

Colocar ambos en esta carpeta antes de ejecutar `main.ipynb`.
